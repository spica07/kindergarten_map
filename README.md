# 서울 유치원 지도

서울시 유치원을 지도에서 찾아보는 정적 웹앱입니다. `index.html` 한 장으로 동작하며 빌드 과정이 없습니다.

## 데이터 출처

- [서울 열린데이터광장 「서울시 유치원 일반현황」](https://data.seoul.go.kr/dataList/OA-20566/A/1/datasetView.do)
- 좌표는 주소 기반 카카오 로컬 API 지오코딩으로 보완
- 학급 편성·정원 등은 조사 시점 기준이며 지원 전 재확인이 필요합니다.

## 기술 스택

- Leaflet + OpenStreetMap 타일
- 빌드 없는 정적 HTML/CSS/JS, GitHub Pages(main 브랜치) 배포

## 안내 문구

본 사이트는 정보 공유 목적의 비영리 제공입니다. 무단 복제·재배포를 금합니다. 공공데이터를 가공해 제작했으며,
표시되는 기관명·상표 등의 권리는 해당 권리자에게 있습니다. 위치 정보는 근사치일 수 있으니 참고용으로만 이용하세요.
